App\bind (host, port, tls)
Map server to an address and port

Parameters:

  • host string Hostname for server to listen on. Set to '0.0.0.0' or '::' to listen on all available interfaces, '127.0.0.1' to listen on the local-only interface, or a custom interface
  • port number Port for server to listen on. Most systems require this value to be above 1024. Use an iptables redirect to use ports below 1024.
  • tls boolean Whether or not to use TLS for connections, defaults to @config.tls
